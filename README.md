# NexTrace Pro

NexTrace Pro is a browser-based Pega-inspired tracing tool for exploring rule execution, sequence flow, filtering, and AI-assisted remediation.

## Features

- Full rule-family filtering with dynamic type discovery from imported traces
- Sequential call diagram with thread lanes and status highlighting
- AI insight board with heuristic diagnostics and optional live LLM enrichment
- Import support for Pega-style XML and JSON event payloads
- Export of filtered trace sessions for sharing or follow-up analysis
